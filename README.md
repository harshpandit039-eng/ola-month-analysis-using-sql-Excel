#  Ola July Month Data Analysis

## Project Overview
This project presents a complete business analysis of Ola ride data for the month of July.  
The objective was to analyze ride trends, revenue performance, cancellations, and customer ratings to generate meaningful business insights.

SQL was used for data extraction, cleaning, and analysis, while Power BI was used to build interactive dashboards.

---

##  Tools & Technologies
- SQL (Data Cleaning & Querying)
- Excel (Initial Data Handling)
- Power BI (Dashboard Development & Visualization)

---

## Dataset Details
The dataset contains:
- Ride ID
- Booking Date & Time
- Vehicle Type
- Booking Status
- Cancellation Details
- Revenue Amount
- Customer Ratings

---

##  Analysis Performed

### 1️ Revenue Analysis
- Total monthly revenue
- Revenue by vehicle type
- Revenue trend over time

### 2️ Cancellation Analysis
- Total cancellations
- Cancellation rate (%)
- Reasons for cancellation

### 3️ Vehicle Type Performance
- Most booked vehicle type
- Revenue comparison by vehicle

### 4️ Ratings Analysis
- Average customer rating
- Rating distribution

### 5️ Peak Time Analysis
- Peak booking hours
- High demand periods

---

##  Key Business Insights
- Identified peak booking hours for operational optimization
- Determined highest revenue generating vehicle type
- Calculated overall cancellation percentage
- Analyzed customer satisfaction trends

---

##  Conclusion
This project demonstrates practical application of SQL queries and Power BI dashboards to solve real-world business problems and extract actionable insights.
